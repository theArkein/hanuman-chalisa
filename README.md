# हनुमान चालीसा

A clean, beautiful, and simple single-page website for reading the Hanuman Chalisa — the 40-verse devotional hymn by Goswami Tulsidas.

**Live site:** [हनुमान चालीसा](https://thearkein.github.io/hanuman-chalisa/)

## Features

- Full text of all 40 chaupais + opening and closing dohas in Devanagari
- Per-line Hindi & English meanings for every verse and doha
- Overall meaning toggle for each doha and chaupai
- Language toggle (हि / EN) to switch between Hindi and English meanings
- Flexible font size controls (A− / A+)
- Auto-scroll with adjustable speed (0.25× – 4×)
- Settings widget (scroll + language + font size) — always-visible trigger, expands on hover/tap
- Reading progress bar
- Smooth fade-in animations
- Fully responsive (mobile-friendly)

## Run locally

With live reload (auto-refreshes on save):

```bash
npx live-server --port=8000
# open http://127.0.0.1:8000
```

No dependencies, no build step — just open `index.html` in a browser.
